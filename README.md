# Guruar Events Celebration Platform

A modern, component-based event management platform built with Next.js. This project focuses on providing a premium, editorial-style experience for planning and managing celebratory events.

## Features

- **Next.js App Router**: Utilizing the latest React features for performance and SEO.
- **Modern UI Components**: Custom-built components designed for a high-end, celebratory feel.
- **Responsive Design**: Optimized for both desktop and mobile experiences.
- **Editorial Layouts**: Intentional asymmetry and sophisticated layering for a unique visual identity.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- **Framework**: Next.js 15+
- **Library**: React 19
- **Styling**: Vanilla CSS (Premium Tonalism)
- **Tooling**: ESLint

## Deployment

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).
